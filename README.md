# webbuilder

## Goal

The application can be used for create website.

## Project Setup

Install Libs

```
npm install
```

Build Project

```
npm run build
```

## Note

The application suppost CommonJS and ESM.

## Useful Commands

Check typescript version

```
npx tsc --version
```
